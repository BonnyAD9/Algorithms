# Arrays

## Reverse
These algorithms reverse array.

```C
void reverse(float* arr, int length);
void recursiveReverse(float* arr, int length);
```

- `reverse`
    - Time: O(n)
    - Memory: O(1)
- `recursiveReverse`
    - Time: O(n)
    - Memory: O(1)
    - TCO: yes
